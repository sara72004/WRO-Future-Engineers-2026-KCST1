# Phase 3 – Laser Sensor Solution

## Solution

The system was redesigned using three laser sensors:

* Front sensor → detects walls
* Left sensor → measures left distance
* Right sensor → measures right distance

## Logic

1. Detect wall ahead
2. Compare left and right distances
3. Turn toward the side with more space

## Advantage

* Works regardless of floor color
* More reliable at high speeds
* Better decision making

## Result

Navigation became significantly more consistent and stable.
