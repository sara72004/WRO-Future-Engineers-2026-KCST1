# Phase 4 – Mechanical Improvements

## Problem

The original front wheels (yellow wheels) caused:

* Shaking
* Uneven movement
* Reduced stability

## Solution

* Replaced front wheels with same type as rear wheels
* Ensured consistent friction across all wheels

## New Issue

* Axle became loose inside the wheel

## Planned Fix

* 3D printed custom axle (preferred)
* Backup: secure with adhesive

## Result

Improved stability and smoother movement.
